# Image Tagging Agent

LangGraph agent package: graph definition, nodes, prompts, schemas, and taxonomy for image analysis and tagging.

## Contents

| File | Description |
|------|-------------|
| `image_tagging.py` | Compiled graph export |
| `graph_builder.py` | Graph construction |
| `configuration.py` | Runtime config |
| `settings.py` | Env vars loader |
| `taxonomy.py` | Tag taxonomy enums |
| `nodes/` | Graph node implementations |
| `prompts/` | Prompt templates |
| `schemas/` | State and data models |
| `tools/` | Agent-specific tools |
