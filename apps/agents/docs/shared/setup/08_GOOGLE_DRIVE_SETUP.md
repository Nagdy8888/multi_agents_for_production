# Google Drive folder (image tagger)

> **Prev:** [07_GMAIL_SETUP.md](07_GMAIL_SETUP.md) | **Index:** [00_SETUP_INDEX.md](00_SETUP_INDEX.md) | **Next:** [09_GAS_TRIGGERS.md](09_GAS_TRIGGERS.md)

1. Create a folder for inbound product images.
2. Open folder; URL contains `.../folders/<FOLDER_ID>`.
3. Set GAS Script Property **`IMAGE_DRIVE_FOLDER_ID`** to that ID.

> **NEXT:** Return to [05_GAS_SCRIPT_PROPERTIES.md](05_GAS_SCRIPT_PROPERTIES.md) and confirm `IMAGE_DRIVE_FOLDER_ID` is set.

The trigger `processNewImages` + `installImageTrigger` are installed in guide **09**.
