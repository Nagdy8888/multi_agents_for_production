# Media (screenshots)

Screenshots for the root README. Save your captures with these names:

| File | Captures |
|------|----------|
| `01-home-before-upload.png` | Home page before uploading an image (empty / dropzone state). |
| `02-analyzing.png` | Screen while the image is being analyzed (e.g. processing overlay). |
| `03-analysis-result.png` | Result after analysis: product image + AI tags, confidence, optional “Saved” badge. |
| `04-search-page.png` | Search page: filters and results grid (and optional detail modal if you prefer). |
| `05-search-filters.png` | Search page with filters in use (e.g. filter sidebar with one or more tags selected, filtered results). |

Use PNG or JPG; if you use JPG, update the image extensions in the root [README.md](../../README.md) Screenshots section.
